Privacy Policy

Last updated: February 14, 2026

Overview

Internet Usage Tracker is a browser extension that tracks the time you spend on websites. This policy explains what data is collected and how it is used.

Data Collected

The extension collects the following information about your browsing activity:

  • URLs and domains of websites you visit
  • Page titles of visited pages
  • Time spent on each page (start time, end time, duration)
  • Device information (device type, OS, browser)
  • Device ID (randomly generated UUID, not linked to your identity)

Data Storage

  • Local Storage: Data is stored locally in your browser until synced
  • Remote Storage: If configured, data syncs to a Supabase database that you control
  • Retention: Local archives are automatically deleted after 30 days (configurable)

Data Sharing

Your data is never sold or shared with third parties. Data is only sent to:

  • The Supabase endpoint you configure in the extension settings
  • No analytics, advertising, or tracking services receive your data

User Control

You have full control over your data:

  • Configure which API endpoint receives your data (or disable syncing entirely)
  • Export all collected data as JSON from the settings page
  • Delete all local data at any time from the settings page
  • Uninstall the extension to stop all data collection

Access Control

This extension uses an allowlist system. Only pre-approved users can sync data to the remote server. To request access, email dhondpratyay@gmail.com.

Security

  • All data transmission uses HTTPS encryption
  • API keys are stored locally in your browser's secure storage
  • The remote database uses Row Level Security (RLS) to isolate user data

Contact

Email: dhondpratyay@gmail.com

Changes

This privacy policy may be updated occasionally. Significant changes will be communicated through extension updates.
